# PlaywrightAutomationLearning

Example Playwright scripts and exercises used for learning Playwright.

Getting started

1. Install dependencies:

```bash
npm install
```

2. Install browsers required by Playwright:

```bash
npx playwright install
```

3. Run tests (if using Playwright Test runner):

```bash
npm test
```
